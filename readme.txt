== Description ==

When selecting a child category in a post, this plugin automatically selects its parent categories.

For example with the categories:

[ ] Sport
    [ ] Football
        [ ] England

Checking England will result in all 3 being checked, like so:

[X] Sport
    [X] Football
        [X] England

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `categoryHierarchy.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
